# SimuSoul - Firebase Studio

This is a NextJS starter project built with Firebase Studio integration.

## Getting Started

To get started with development, take a look at `src/app/page.tsx`.

## Features
- NextJS framework
- Firebase integration
- Modern React development

## Android App

SimuSoul is also available as a native Android app. The Android app wraps the web application in a fullscreen WebView for a native experience.

See [android/README.md](./android/README.md) for build instructions.
